librepfunc-1.5.0

Introduction to librepfunc

librepfunc provides a set of C++ classes and utilities for building multimedia tools.

This package is known to build and work properly using an LFS-11.1 platform.

Package Information

  • Download (HTTP): https://github.com/wirbel-at-vdr-portal/librepfunc/archive/refs/tags/1.5.0.tar.gz

  • Download MD5 sum: b2cb6a65f7a2a0efd5200a175e8628fc

  • Download size: 0.025 MB

  • Estimated disk space required: 9 MB

  • Estimated build time: 0.1 SBU (using parallelism=4)

Installation of librepfunc

Install librepfunc by running the following commands: 

make

This package does not come with a testsuite.

Now, as the root user: 

make -j1 install
